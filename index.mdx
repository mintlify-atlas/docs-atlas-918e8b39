# MCP Kotlin SDK

The official Kotlin SDK for the [Model Context Protocol](https://modelcontextprotocol.io) enables Kotlin applications targeting JVM, Native, JS, and Wasm to implement MCP clients and servers using a standardized protocol interface.

Build context-aware AI applications with type-safe APIs, multiple transport options, and first-class coroutine support.

## Example Code

Get started with a minimal MCP server:

```kotlin
val mcpServer = Server(
    serverInfo = Implementation("example-server", "1.0.0"),
    options = ServerOptions(
        capabilities = ServerCapabilities(
            tools = ServerCapabilities.Tools(listChanged = true)
        )
    )
)

mcpServer.addTool(
    name = "echo",
    description = "Echo text back to the user"
) { request ->
    val text = request.arguments?.get("text")?.jsonPrimitive?.content ?: ""
    CallToolResult(content = listOf(TextContent("Echo: $text")))
}

embeddedServer(CIO, port = 3000) {
    mcp { mcpServer }
}.start(wait = true)
```
