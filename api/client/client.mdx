The `Client` class provides a high-level implementation for connecting to MCP servers and calling their exposed capabilities. It automatically handles protocol initialization, capability negotiation, and request/response correlation.

## Package

```kotlin
io.modelcontextprotocol.kotlin.sdk.client
```

## Quick Start Helper

### mcpClient

A convenience function that creates a client and connects to a server in one call.

```kotlin
suspend fun mcpClient(
    clientInfo: Implementation,
    clientOptions: ClientOptions = ClientOptions(),
    transport: Transport
): Client
```

<ParamField path="clientInfo" type="Implementation" required>
  Information about this client implementation (name and version)
</ParamField>

<ParamField path="clientOptions" type="ClientOptions">
  Configuration options including capabilities
</ParamField>

<ParamField path="transport" type="Transport" required>
  The transport to use for communication
</ParamField>

**Returns:** A connected `Client` instance ready to use

**Example:**

```kotlin
val client = mcpClient(
    clientInfo = Implementation("my-client", "1.0.0"),
    transport = StdioClientTransport(processInput, processOutput)
)

// Client is already connected and ready to use
val tools = client.listTools()
```

<Note>
  This helper is marked with `@ExperimentalMcpApi` and combines client creation and connection in a single call.
</Note>

## Methods

### connect

Connects to a server using the given transport and performs initialization handshake.

```kotlin
override suspend fun connect(transport: Transport)
```

<ParamField path="transport" type="Transport" required>
  The transport to use for communication
</ParamField>

**Example:**

```kotlin
val client = Client(
    clientInfo = Implementation("my-client", "1.0.0"),
    options = ClientOptions(
        capabilities = ClientCapabilities(
            roots = ClientCapabilities.Roots(listChanged = true)
        )
    )
)

val transport = StdioClientTransport(processInput, processOutput)
client.connect(transport)

// Client is now ready to use
println("Connected to ${client.serverVersion?.name}")
```

### Tool Operations

#### callTool (with parameters)

Calls a tool with dynamic arguments.

```kotlin
suspend fun callTool(
    name: String,
    arguments: Map<String, Any?>,
    meta: Map<String, Any?> = emptyMap(),
    options: RequestOptions? = null
): CallToolResult
```

<ParamField path="name" type="String" required>
  Name of the tool to call
</ParamField>

<ParamField path="arguments" type="Map<String, Any?>" required>
  Tool arguments as a map
</ParamField>

<ParamField path="meta" type="Map<String, Any?>">
  Optional metadata (must follow MCP meta key format)
</ParamField>

<ParamField path="options" type="RequestOptions">
  Optional request options (timeout, progress callback)
</ParamField>

**Example:**

```kotlin
val result = client.callTool(
    name = "get_weather",
    arguments = mapOf(
        "location" to "San Francisco",
        "units" to "fahrenheit"
    ),
    options = RequestOptions(timeout = 30.seconds)
)

result.content.forEach { content ->
    when (content) {
        is TextContent -> println(content.text)
        is ImageContent -> saveImage(content.data)
        else -> {}
    }
}
```

#### callTool (with request)


**Example:**

```kotlin

    request: ListPromptsRequest = ListPromptsRequest(),
    options: RequestOptions? = null

```

**Example:**

```kotlin

    }
```

## Complete Example

```kotlin
import io.modelcontextprotocol.kotlin.sdk.client.*
import io.modelcontextprotocol.kotlin.sdk.types.*
import kotlinx.coroutines.runBlocking
import kotlin.time.Duration.Companion.seconds

fun main() = runBlocking {
    // Create client
    val client = Client(
        clientInfo = Implementation("my-app", "1.0.0"),
        options = ClientOptions(
            capabilities = ClientCapabilities(
                roots = ClientCapabilities.Roots(listChanged = true)
            )
        )
    )
    
    // Connect
    val process = ProcessBuilder("mcp-server").start()
    val transport = StdioClientTransport(
        input = process.inputStream.asSource(),
        output = process.outputStream.asSink()
    )
    
    client.connect(transport)
    
    println("Connected to ${client.serverVersion?.name} ${client.serverVersion?.version}")
    client.serverInstructions?.let { println("Instructions: $it") }
    
    // List available tools
    val tools = client.listTools()
    println("\nAvailable tools:")
    tools.tools.forEach { tool ->
        println("  - ${tool.name}: ${tool.description}")
    }
    
    // Call a tool
    val result = client.callTool(
        name = "get_weather",
        arguments = mapOf("location" to "San Francisco"),
        options = RequestOptions(timeout = 10.seconds)
    )
    
    println("\nWeather result:")
    result.content.forEach { content ->
        when (content) {
            is TextContent -> println(content.text)
            else -> {}
        }
    }
    
    // Cleanup
    client.close()
    process.destroy()
}
```
