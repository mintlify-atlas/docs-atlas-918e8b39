title: Server
description: High-level server implementation for exposing MCP capabilities
---

The `Server` class provides a high-level implementation for creating MCP servers that can expose tools, prompts, and resources to clients. It manages multiple client sessions and automatically handles protocol initialization and capability negotiation.

## Package

```kotlin
io.modelcontextprotocol.kotlin.sdk.server
```

## Class Definition

```kotlin
open class Server(
    protected val serverInfo: Implementation,
    protected val options: ServerOptions,
    protected val instructionsProvider: (() -> String)? = null,
    block: Server.() -> Unit = {}
)
```

<ParamField path="serverInfo" type="Implementation" required>
  Information about this server implementation (name and version)
</ParamField>

<ParamField path="options" type="ServerOptions" required>
  Configuration options including capabilities
</ParamField>

<ParamField path="instructionsProvider" type="() -> String">
  Optional provider for instructions from the server to the client. Called each time a new session starts to support dynamic instructions.
</ParamField>

<ParamField path="block" type="Server.() -> Unit">
  Configuration block executed during initialization
</ParamField>

### Tool Management

#### addTool (with Tool object)

```kotlin
fun addTool(
    tool: Tool,
    handler: suspend ClientConnection.(CallToolRequest) -> CallToolResult
)
```

<ParamField path="tool" type="Tool" required>
  Tool definition
</ParamField>

<ParamField path="handler" type="suspend ClientConnection.(CallToolRequest) -> CallToolResult" required>
  Suspend function that executes the tool
</ParamField>

**Example:**

```kotlin
server.addTool(
    tool = Tool(
        name = "calculator",
        description = "Performs basic math operations",
        inputSchema = ToolSchema(
            properties = buildJsonObject {
                put("operation", buildJsonObject {
                    put("type", "string")
                    put("enum", buildJsonArray {
                        add("add")
                        add("subtract")
                    })
                })
                put("a", buildJsonObject { put("type", "number") })
                put("b", buildJsonObject { put("type", "number") })
            },
            required = listOf("operation", "a", "b")
        )
    )
) { request ->
    val args = request.params.arguments
    val op = args["operation"]?.jsonPrimitive?.content
    val a = args["a"]?.jsonPrimitive?.double ?: 0.0
    val b = args["b"]?.jsonPrimitive?.double ?: 0.0
    
    val result = when (op) {
        "add" -> a + b
        "subtract" -> a - b
        else -> error("Unknown operation")
    }
    
    CallToolResult(
        content = listOf(TextContent(result.toString()))
    )
}
```

            )

## Complete Example

```kotlin
import io.modelcontextprotocol.kotlin.sdk.server.*
import io.modelcontextprotocol.kotlin.sdk.types.*
import kotlinx.io.asSource
import kotlinx.io.asSink

fun main() = runBlocking {
    val server = Server(
        serverInfo = Implementation(
            name = "example-server",
            version = "1.0.0"
        ),
        options = ServerOptions(
            capabilities = ServerCapabilities(
                tools = ServerCapabilities.Tools(listChanged = true),
                resources = ServerCapabilities.Resources(subscribe = false),
                prompts = ServerCapabilities.Prompts()
            )
        ),
        instructions = "This is an example MCP server"
    )

    // Add a simple tool
    server.addTool(
        name = "echo",
        description = "Echoes back the input"
    ) { request ->
        val message = request.params.arguments["message"]?.jsonPrimitive?.content ?: ""
        CallToolResult(
            content = listOf(TextContent("Echo: $message"))
        )
    }

    // Add a prompt
    server.addPrompt(
        name = "greeting",
        description = "A friendly greeting"
    ) { request ->
        GetPromptResult(
            messages = listOf(
                PromptMessage(
                    role = Role.User,
                    content = TextContent("Hello! How can I help you today?")
                )
            )
        )
    }

    // Create session with STDIO transport
    val transport = StdioServerTransport(
        System.`in`.asSource(),
        System.out.asSink()
    )
    
    val session = server.createSession(transport)
    
    // Keep server running
    awaitCancellation()
}
```
