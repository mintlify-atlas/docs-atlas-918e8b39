This guide walks you through creating a simple MCP client and server using real examples from the SDK.

## Prerequisites

- Kotlin 2.2+ installed
- Gradle project set up
- MCP SDK dependencies added (see [Installation](/installation))

## Creating a Client

Let's build a client that connects to an MCP server and lists available tools.

<Steps>
  <Step title="Import required dependencies">
    ```kotlin
    import io.ktor.client.HttpClient
    import io.ktor.client.plugins.sse.SSE
    import io.modelcontextprotocol.kotlin.sdk.client.Client
    import io.modelcontextprotocol.kotlin.sdk.client.StreamableHttpClientTransport
    import io.modelcontextprotocol.kotlin.sdk.types.Implementation
    import kotlinx.coroutines.runBlocking
    ```
  </Step>

  <Step title="Create and configure the client">
    ```kotlin
    fun main(args: Array<String>) = runBlocking {
        val url = args.firstOrNull() ?: "http://localhost:3000/mcp"

        // Create Ktor HTTP client with SSE support
        val httpClient = HttpClient { install(SSE) }

        // Initialize MCP client
        val client = Client(
            clientInfo = Implementation(
                name = "example-client",
                version = "1.0.0"
            )
        )

        // Create transport for communication
        val transport = StreamableHttpClientTransport(
            client = httpClient,
            url = url
        )

        // Connect to server
        client.connect(transport)

        // List available tools
        val tools = client.listTools().tools
        println("Available tools: ${tools.map { it.name }}")
    }
    ```

    <Note>
      The client uses `Implementation` to identify itself to the server with a name and version.
    </Note>
  </Step>

  <Step title="Call a tool">
    Add tool invocation to interact with server capabilities:

    ```kotlin
    // Call a specific tool
    val result = client.callTool(
        name = "echo",
        arguments = mapOf("text" to "Hello, MCP!")
    )

    result.content.forEach { content ->
        if (content is TextContent) {
            println(content.text)
        }
    }
    ```
  </Step>
</Steps>

### STDIO Client Example

For connecting to servers via process stdin/stdout:

```kotlin
import io.modelcontextprotocol.kotlin.sdk.client.Client
import io.modelcontextprotocol.kotlin.sdk.client.StdioClientTransport
import io.modelcontextprotocol.kotlin.sdk.types.Implementation
import kotlinx.io.asSink
import kotlinx.io.asSource
import kotlinx.io.buffered

fun connectToStdioServer(serverScriptPath: String) {
    // Start the server process
    val process = ProcessBuilder("node", serverScriptPath).start()

    // Setup I/O transport using process streams
    val transport = StdioClientTransport(
        input = process.inputStream.asSource().buffered(),
        output = process.outputStream.asSink().buffered()
    )

    // Create and connect client
    val client = Client(
        clientInfo = Implementation("stdio-client", "1.0.0")
    )
    client.connect(transport)

    // Now use the client...
}
```

## Creating a Server

Now let's build a server that exposes tools and resources.

<Steps>
  <Step title="Import server dependencies">
    ```kotlin
    import io.ktor.server.cio.CIO
    import io.ktor.server.engine.embeddedServer
    import io.modelcontextprotocol.kotlin.sdk.server.Server
    import io.modelcontextprotocol.kotlin.sdk.server.ServerOptions
    import io.modelcontextprotocol.kotlin.sdk.server.mcp
    import io.modelcontextprotocol.kotlin.sdk.types.*
    import kotlinx.serialization.json.buildJsonObject
    import kotlinx.serialization.json.put
    ```
  </Step>

  <Step title="Create and configure the server">
    ```kotlin
    fun main(args: Array<String>) {
        val port = args.firstOrNull()?.toIntOrNull() ?: 3000

        // Initialize MCP server
        val mcpServer = Server(
            serverInfo = Implementation(
                name = "example-server",
                version = "1.0.0"
            ),
            options = ServerOptions(
                capabilities = ServerCapabilities(
                    tools = ServerCapabilities.Tools(listChanged = true)
                )
            )
        )
    }
    ```
  </Step>

  <Step title="Add a tool">
    Register a tool that the server will expose:

    ```kotlin
    mcpServer.addTool(
        name = "echo",
        description = "Echoes back the input text",
        inputSchema = ToolSchema(
            properties = buildJsonObject {
                put("text", buildJsonObject { 
                    put("type", "string")
                    put("description", "Text to echo")
                })
            },
            required = listOf("text")
        )
    ) { request ->
        val text = request.arguments?.get("text")?.jsonPrimitive?.content 
            ?: "(empty)"
        CallToolResult(
            content = listOf(TextContent(text = "Echo: $text"))
        )
    }
    ```
  </Step>

  <Step title="Start the server">
    Launch the server with Ktor:

    ```kotlin
    embeddedServer(CIO, host = "127.0.0.1", port = port) {
        mcp {
            mcpServer
        }
    }.start(wait = true)

    println("Server running on http://localhost:$port")
    ```

    <Tip>
      The `mcp` DSL automatically configures SSE endpoints and CORS for client connections.
    </Tip>
  </Step>
</Steps>

### Complete Server Example

Here's a working server with multiple features:

<CodeGroup>
  ```kotlin Server with Tools
  fun configureServer(): Server {
      val server = Server(
          Implementation("mcp-kotlin-server", "0.1.0"),
          ServerOptions(
              capabilities = ServerCapabilities(
                  tools = ServerCapabilities.Tools(listChanged = true),
                  resources = ServerCapabilities.Resources(
                      subscribe = true, 
                      listChanged = true
                  )
              )
          )
      )

      // Add a tool
      server.addTool(
          name = "greet",
          description = "Greet a user by name"
      ) { request ->
          val name = request.arguments?.get("name")?.jsonPrimitive?.content
          CallToolResult(
              content = listOf(TextContent("Hello, ${name ?: "World"}!"))
          )
      }

      // Add a resource
      server.addResource(
          uri = "config://settings",
          name = "Server Settings",
          description = "Current server configuration",
          mimeType = "application/json"
      ) { request ->
          ReadResourceResult(
              contents = listOf(
                  TextResourceContents(
                      text = """{ "version": "1.0.0" }""",
                      uri = request.uri,
                      mimeType = "application/json"
                  )
              )
          )
      }

      return server
  }
  ```

  ```kotlin Weather Server (STDIO)
  fun runWeatherServer() {
      val server = Server(
          Implementation("weather", "1.0.0"),
          ServerOptions(
              capabilities = ServerCapabilities(
                  tools = ServerCapabilities.Tools(listChanged = true)
              )
          )
      )

      server.addTool(
          name = "get_forecast",
          description = "Get weather forecast for coordinates",
          inputSchema = ToolSchema(
              properties = buildJsonObject {
                  putJsonObject("latitude") { put("type", "number") }
                  putJsonObject("longitude") { put("type", "number") }
              },
              required = listOf("latitude", "longitude")
          )
      ) { request ->
          val lat = request.arguments?.get("latitude")?.jsonPrimitive?.doubleOrNull
          val lon = request.arguments?.get("longitude")?.jsonPrimitive?.doubleOrNull
          
          if (lat == null || lon == null) {
              return@addTool CallToolResult(
                  content = listOf(TextContent("Invalid coordinates"))
              )
          }

          // Fetch weather data...
          CallToolResult(
              content = listOf(TextContent("Forecast for $lat,$lon"))
          )
      }

      // Use STDIO transport
      val transport = StdioServerTransport(
          System.`in`.asSource().buffered(),
          System.out.asSink().buffered()
      )

      runBlocking {
          val session = server.createSession(transport)
          session.onClose { println("Server closed") }
          awaitCancellation()
      }
  }
  ```
</CodeGroup>

## Testing Your Setup

<Steps>
  <Step title="Start the server">
    Run your server application:

    ```bash
    ./gradlew run
    ```
  </Step>

  <Step title="Connect with MCP Inspector">
    Use the official MCP Inspector to test your server:

    ```bash
    npx -y @modelcontextprotocol/inspector
    ```

    Connect to `http://localhost:3000` in the Inspector UI.
  </Step>

  <Step title="Test with your client">
    Run your client application pointing to your server:

    ```bash
    ./gradlew runClient --args="http://localhost:3000"
    ```
  </Step>
</Steps>

## Transport Options

The SDK supports multiple transport mechanisms:

<AccordionGroup>
  <Accordion title="STDIO - Process Communication">
    Perfect for editor plugins and CLI tools:

    ```kotlin
    // Client
    val transport = StdioClientTransport(
        input = process.inputStream.asSource().buffered(),
        output = process.outputStream.asSink().buffered()
    )

    // Server
    val transport = StdioServerTransport(
        inputStream = System.`in`.asSource().buffered(),
        outputStream = System.out.asSink().buffered()
    )
    ```
  </Accordion>

  <Accordion title="Streamable HTTP - Recommended for Remote">
    Best for remote deployments with SSE streaming:

    ```kotlin
    // Client
    val transport = StreamableHttpClientTransport(
        client = HttpClient { install(SSE) },
        url = "http://localhost:3000/mcp"
    )

    // Server (Ktor)
    embeddedServer(CIO, port = 3000) {
        mcpStreamableHttp(path = "/mcp") {
            MyServer()
        }
    }.start(wait = true)
    ```
  </Accordion>

  <Accordion title="WebSocket - Full Duplex">
    For low-latency, bidirectional communication:

    ```kotlin
    // Client
    val transport = WebSocketClientTransport(
        client = HttpClient { install(WebSockets) },
        url = "ws://localhost:3000/mcp"
    )

    // Server configuration similar to SSE
    ```
  </Accordion>
</AccordionGroup>

<Warning>
  Each transport requires appropriate Ktor dependencies. See the [Installation](/installation) guide for details.
</Warning>
